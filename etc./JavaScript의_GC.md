## JavaScript의 GC

### 가비지 컬렉션(GC)
프로그래밍 언어에서 동적으로 할당된 메모리 중 더 이상 사용되지 않는 영역, 즉 '쓰레기'를 자동으로 찾아 제거하여 메모리 공간을 확보하는 자동 메모리 관리 기법


**⦿ 메모리 생존 주기**

**1. 메모리 할당**

원시 타입의 값들은 스택 영역에 저장, 참조 타입의 값은 힙 영역에 저장되며 그 주소값은 스택 영역에 저장된다. 변수 식별자는 스택의 실행 컨텍스트의 렉시컬 환경에 저장!

**2. 할당된 메모리 사용(참조)**
